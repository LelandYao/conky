操作方法
1. 在linux安装conky和conky-lua 
  sudo apt-get install conky-all (ubuntu以及基于debain的这样安装，其他的请自行百度)
2. 在当前家目录创建.Conky文件夹
3. 将文件clone到该文件夹
4. 运行startconky.sh文件  ./startconky.sh
